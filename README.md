# Gin Helloworld

Helloworld webapp based on the go framework Gin.

# Prerequisites

- Go (1.15 or above )

# Getting Started

- Download modules `go mod download`
- Run the app `go run main.go`
- Test it `curl http://localhost:8080/ping`
